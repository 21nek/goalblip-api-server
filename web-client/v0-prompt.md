6. CTA Section:
   - "Hemen Ücretsiz Dene!"
   - App Store and Play Store download buttons with badges

7. Footer:
   - Logo + short description

- Logo (left)
- Navigation: Home | Privacy Policy | Support (center)

- Text colors: #f8fafc (primary), #cbd5f5 (secondary), #94a3b8 (tertiary)
- Icons: Modern, minimal, gradient accents

FEATURES:
- Multi-language support (TR, EN, ES, DE) - switch via URL param ?lang=tr
- Smooth scroll navigation
- Responsive hamburger menu for mobile
- Scroll animations (fade-in, slide-up on scroll)
- Lazy loading images

Make it clean, professional, and match a mobile app's design identity. Use semantic HTML, CSS3, and vanilla JavaScript for language switching and smooth scroll.
